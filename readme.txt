== Description ==

Provides a GPA calculator for students.

== Installation ==

1. Upload the files to the `/wp-content/plugins/gpa-calculator/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the [gpa-calculator] short code in any page or post
4. Place the text in intro.txt in the page before the short code
